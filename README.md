# Portfolio
This is a portfolio website boostrapped with `npm create vite@latest ./ -- --template react`

## Run the app
- Clone the repository.
```
git clone https://github.com/OMKAR-KALEPU/portfolio
```

- Install dependencies.
```
npm install
```

- Start the vite app.
```
npm run dev
```

- The App is live.
```
 http://localhost:5173/
```
